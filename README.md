# Investment Platform AI-Native

基于投资管理领域的思考，探索 AI 原生体验的应用形态。

## 项目定位

这是一个**抛弃型 PoC（Proof of Concept）**，目标不是构建生产系统，而是快速验证理念：

- 所有原始业务规则均已**脱敏处理**，仅保留逻辑结构
- 借助**公有云服务**与**本地资源**快速搭建，优先验证速度
- 聚焦 AI 原生交互模式的可行性，而非工程完整性

## 验证重点

虽然是抛弃型原型，但本次验证有明确的优先级：

| 优先级 | 方向 | 策略 |
|--------|------|------|
| **高** | 用户交互体验 | 完整验证，追求真实体感 |
| **高** | AI 智能体验 | 完整验证，覆盖核心智能场景 |
| **高** | 安全性 | 完整验证，不因原型性质降低标准 |
| **适度** | 大规模调用 | 可裁剪，但做出设计预留，确保架构可扩展 |

> 原型可以丢弃，但验证结论必须可信。交互、智能、安全三个维度不打折扣；性能与规模维度做合理裁剪，同时在架构上预留扩展路径，确保结论可以指导后续生产系统的设计。

## 核心探索方向

- 投资管理场景下的 AI 原生交互体验
- 业务规则与 AI 能力的融合方式
- 从传统 CRUD 到智能决策辅助的路径验证

## 说明

本项目为实验性质，不包含任何真实业务数据或敏感规则。代码与架构可能随探索方向调整而大幅变动。
